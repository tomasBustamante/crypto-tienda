## Pre-requisitos

- truffle
- node (usamos la versión lts con nvm)

## Instalación

```console
cd client
npm i
```

## Ejecución local

### Server side

```console
truffle develop
```

Dentro de truffle:

```console
truffle compile
truffle migrate --reset
```

### Cliente

```console
cd client
npm run start
```
